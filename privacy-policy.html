Stock Price Widget Privacy Policy

Last updated: 2026-03-18

Stock Price Widget provides stock and cryptocurrency quote widgets on webpages selected by the user.

Overview

The extension allows users to search symbols, create widgets, and display real-time price updates on webpages. To provide this functionality, the extension runs on user-opened webpages, stores widget layout locally in the browser, and connects to a developer-operated quote server for search and quote delivery.

Data We Handle

  • Widget configuration chosen by the user, such as symbol, display name, position, width, scale, and minimal mode preference
  • Quote lookup requests and realtime quote subscription data sent to the quote server
  • Basic technical request and connection logs used to operate, secure, and debug the service
  • Limited page context required to render widgets on the current webpage selected by the user

How We Use Data

  • Search supported stock and cryptocurrency symbols
  • Stream realtime quote data to the extension
  • Save widget layout and user preferences locally in the browser
  • Prevent abuse, apply rate limits, and diagnose service issues

Data Storage

  • Widget layout and user preferences are stored locally in Chrome storage on the user's device
  • Quote data may be cached temporarily by the extension and quote server to deliver current prices efficiently
  • Server logs may temporarily contain technical metadata such as IP address, request timing, and rate-limit events

Data Sharing

We share data only with services necessary to operate the extension:

  • Korea Investment & Securities Open API for domestic stock quotes
  • Gate.io public market data APIs for cryptocurrency quotes
  • Hosting and infrastructure providers used to run the quote server

We do not sell user data. We do not share user data for advertising or marketing purposes.

Advertising and Profiling

  • We do not use user data for personalized advertising
  • We do not sell browsing activity
  • We do not use webpage data for unrelated analytics, profiling, or marketing

Security

  • Production traffic between the extension and the quote server is intended to use HTTPS/WSS
  • We use service protections such as rate limiting and operational monitoring to reduce abuse

User Control

Users can remove widgets, clear extension data stored in the browser, or uninstall the extension at any time. Uninstalling the extension stops further extension-side data handling in the browser.
